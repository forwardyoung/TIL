# 조건문

>  특정 조건을 기준으로 무언가를 결정할 때 사용하는 문법

![](https://github.com/forwardyoung/TIL/blob/master/Java/Java.assets/6-1.png)

## if문

> 특정 동작의 수행 여부를 결정할 때 사용

```java
if ( 덥다면? ) {
    에어컨을 튼다!
}
```

## if-else문

> 둘 중 하나를 선택할 때 사용

```java
if (날씨가 추우면) {
    따뜻한 라떼 주문!
} else {
    시원한 음료 주문!
}
```

## else-if문

> 여러 가지 선택지 중 하나를 고를 때 사용

```java
if ( 90점보다 큼? ) {
    A학점!
} else if ( 80점보다 큼? ) {
    B학점!
} else {
    그 외 C학점!
}
```
